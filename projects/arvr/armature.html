Armature

By: Thomas Suarez, Elliot Mebane and Ryan Reede

Armature is a holographic stencilling tool for stop-motion animators built in 48 hrs at USC's Creating Reality Hackathon (March 2018). The tool was built for Mira Prism in Unity and allows animators to use a digital animation file (ie: FBX) to help guide their animations. When in the headset, the user can step through the animation file frame-by-frame, have that hologram positionally tracked in their workspace, and essentially have a 3D onion-skin with which to guide them as they animate a physical character.
